Frequently Asked Questions

Is HackToolkit safe to use? +
Absolutely. HackToolkit is a configuration utility that does not modify your Windows system files. It focuses on creating the necessary EFI files for your macOS installation. Always back up your data before modifying partitions.
Does it support the latest macOS Sequoia? +
Yes, HackToolkit v1.0.0 includes support for the latest OpenCore patches required for macOS Sequoia, as well as legacy support for older versions.
How often is the kext database updated? +
The internal database syncs with official repositories every time you run the tool (if an internet connection is available), ensuring you always have the latest Lilu, VirtualSMC, and WhateverGreen versions.
Can I use this for AMD Ryzen builds? +
Yes! HackToolkit has a dedicated AMD path that automatically applies necessary kernel patches and selects the correct kexts for Ryzen and Threadripper systems.
